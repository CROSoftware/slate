    Authentication

    oAuth2 authentication.

    To obtain an access token, your CRO API client must complete an OAuth 2.0 flow. The access token must be passed as an [Authorization: Bearer <token>] header with each request.

    • Flow: authorizationCode
      • Authorization URL = https://auth.crosandbox.com/oauth2/auth
      • Token URL = https://auth.crosandbox.com/oauth2/token
    Scope Scope Description
    api CRO API access
    • Flow: implicit
      • Authorization URL = https://auth.crosandbox.com/oauth2/auth
    Scope Scope Description
    api CRO API access

    • Flow: password

      • Token URL = https://auth.crosandbox.com/oauth2/token
    Scope Scope Description
    api CRO API access

    • Flow: clientCredentials
